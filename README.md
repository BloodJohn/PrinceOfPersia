# PrinceOfPersia
HTML retro platformer

Создай игру в стиле Prince of Persia на HTML5 Canvas с использованием JavaScript.

Требования:
- Ретро графика в стиле 8-bit/16-bit
- Персонаж принц с анимацией бега, прыжков и атаки
- Платформерная механика с гравитацией
- Ловушки: шипы, падающие камни, движущиеся платформы
- Простое управление: стрелки для движения, пробел для прыжка, Z для атаки
- Несколько уровней с увеличивающейся сложностью
- Система здоровья и смертей
- Враги с простым ИИ
- Фоновая музыка в ретро стиле
- Звуковые эффекты для действий
- Счетчик очков и времени

Технические детали:
- Используй чистый JavaScript без фреймворков
- Оптимизация для 60 кадров в секунду
- Адаптивный дизайн для разных размеров экрана
- Простая система сохранения прогресса
- Меню паузы и рестарта уровня

Стиль графики:
- Пиксельная графика с ограниченной цветовой палитрой
- Анимации спрайтов для персонажа и врагов
- Фоновые декорации в персидском стиле
- Эффекты частиц для ударов и смертей
